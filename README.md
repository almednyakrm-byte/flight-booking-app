# تطبيق حجز طيران مع إدارة الرحلات والتسعير والسياسات
==============================================

## ملخص المشروع
----------------

تطبيق حجز طيران مع إدارة الرحلات والتسعير والسياسات هو تطبيق يهدف إلى تسهيل عملية حجز الرحلات الجوية للمستخدمين، مع إمكانية إدارة الرحلات والتسعير والسياسات المرتبطة بها.

## هيكلية المشروع
-----------------

### مكونات المشروع

* `docker-compose.yml`: يحتوي على تكوين المشروع ووحدات التخزين والخادم.
* `app`: يحتوي على الكود المصدر للتطبيق.
* `database`: يحتوي على تكوين قاعدة البيانات.
* `docker`: يحتوي على الملفات الدوكير للخادم والوحدات التخزينية.

## إعداد وتشغيل المشروع
-------------------------

### إعداد المشروع

1. تأكد من أن لديك Docker و Docker Compose مثبتين على جهازك.
2. افتح المجلد الرئيسي للمشروع في شاشة командية.
3. استخدم الأمر التالي لتشغيل المشروع:

bash
docker-compose up


### تشغيل المشروع

1. بعد تشغيل المشروع، يمكنك الوصول إلى التطبيق عبر الرابط `http://localhost:5000`.
2. يمكنك استخدام أدوات التطوير مثل `curl` أو `Postman` لاختبار الوظائف المختلفة للتطبيق.

## وحدات التطبيق
-----------------

### وحدات البيانات

* `flight`: يحتوي على معلومات حول الرحلات الجوية.
* `user`: يحتوي على معلومات حول المستخدمين.

### أدوار المستخدمين

* `admin`: يمكنه إدارة الرحلات والتسعير والسياسات.
* `user`: يمكنه حجز الرحلات.
